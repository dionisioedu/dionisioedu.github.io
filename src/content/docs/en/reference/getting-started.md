title: Where to Start?
---

If you are just starting now, or if you still plan to move into technology and programming, know that it is very common to feel lost in the middle of so much information.

Here I will list **what you need to learn**, and in which order, so you can start thinking like a real software engineer.

:::note
First of all, in case you have not noticed yet: to stand out in this profession, you will need to study a lot, read a lot, and struggle through many hard problems — so get used to it.
:::

### The Basics

To think like a programmer, you need to understand some abstract concepts that may not make sense in the “real world”, but make perfect sense when thinking like a machine.

Regardless of the area you want to work in, these are the concepts you must understand:
- [Data Types](/en/reference/tipos-de-dados/)
- [Data Structures](/en/reference/estruturas-de-dados/)
- [Programming Logic](/en/reference/logica-de-programacao/)
- [Algorithms](/en/reference/algoritmos/)

This is the first part. So dedicate some time to studying each of these topics, in this order.

If you master this foundation, every new language becomes much easier, and you stop depending on “copy-paste recipes” to solve problems.
