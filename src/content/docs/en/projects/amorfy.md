# Amorfy

## Overview

**Amorfy** is an interactive product that uses a multi-step questionnaire to guide the user toward a personalized result.

## What this project teaches

- step-based flow design
- progressive user experience
- interface clarity for longer forms
- result communication at the end of a journey

## What kind of problem it solves

Projects like this are useful for studying how to keep users engaged during longer interactions, reduce abandonment, and make each step easier to understand.

## What is worth observing

- how the questionnaire structures progression
- how the product turns answers into perceived value
- how interface copy shapes the experience

## Suggested next steps

1. Review the full flow and identify the points with the most friction.
2. Compare it with [Wheel Of List](/en/projects/wheel-of-list/) to see two very different UX models.
3. Use this case as reference when thinking about onboarding, forms, and guided experiences.
