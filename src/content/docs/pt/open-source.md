## Open Source

Projetos que projetei, construí e coloquei em produção. Cada um resolve um problema real.

### Code Royale

Plataforma multiplayer de batalhas de código. Gameplay em tempo real via WebSocket, bots
programáveis, ranking Elo e um banco de desafios com 9 tópicos de programação.

| | |
|---|---|
| **Stack** | Go, Next.js 15, TypeScript, PostgreSQL, Redis, WebSocket, Judge0 |
| **Destaques** | Matchmaking lock-free, IA dos bots com escala Elo, i18n (PT/ES), Cloud Run |
| **Repo** | [github.com/dionisioedu/code-royale](https://github.com/dionisioedu/code-royale) |
| **Ao vivo** | [coderoyale.app](https://coderoyale.app) |

### Wheel of List

Roleta de sorteio com mais de 10 ferramentas — sorteio de nomes, sim/não, dados, times e mais.
Zero dependências, carregamento instantâneo.

| | |
|---|---|
| **Stack** | Vanilla JS, CSS, HTML5 Canvas, GitHub Pages |
| **Destaques** | 37 páginas, otimizado para AdSense, 10 ferramentas + 8 templates + 15 posts |
| **Repo** | [github.com/dionisioedu/wheelofnames](https://github.com/dionisioedu/wheelofnames) |
| **Ao vivo** | [wheeloflist.com](https://wheeloflist.com) |
