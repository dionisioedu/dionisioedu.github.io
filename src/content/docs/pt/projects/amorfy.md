# Amorfy

## Visão geral

O **Amorfy** é um produto interativo que usa um questionário em múltiplas etapas para conduzir o usuário até um resultado personalizado.

## O que este projeto ensina

- desenho de fluxo guiado por etapas
- construção de experiência progressiva
- clareza de interface para formulários longos
- comunicação de resultado no final da jornada

## Tipo de problema que ele resolve

Projetos assim são úteis para estudar como manter o usuário engajado durante uma sequência mais longa de interação, reduzindo abandono e deixando cada etapa mais clara.

## O que vale observar ao navegar

- como o questionário organiza a progressão
- como o produto transforma respostas em percepção de valor
- como o texto da interface influencia a experiência

## Próximas ações para estudo

1. Reflita sobre o fluxo completo e identifique os pontos de maior atrito.
2. Compare com o projeto [Wheel Of List](/pt/projects/wheel-of-list/) para ver dois tipos bem diferentes de UX.
3. Use este case como referência quando pensar em formulários, onboarding e experiências guiadas.
