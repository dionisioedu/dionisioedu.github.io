# Wheel Of List

## Visão geral

O **Wheel Of List** é uma ferramenta web simples e direta para sorteio de nomes em formato de roleta, com recursos extras para sorteio de números e dados.

## O que este projeto ensina

- construção de ferramenta com proposta clara
- uso de animação para reforçar feedback
- experiência rápida e sem fricção
- valor de produtos pequenos, mas bem resolvidos

## Por que ele importa em um portal dev

Nem todo bom projeto precisa ser grande. Ferramentas utilitárias mostram muito bem como pequenas decisões de interface e interação afetam percepção de qualidade.

## O que vale observar ao navegar

- como a ação principal é entendida rapidamente
- como o feedback visual ajuda a sensação de resultado
- como escopo enxuto pode gerar produto útil

## Próximas ações para estudo

1. Pense em como você organizaria os estados da interface.
2. Compare com o [Amorfy](/pt/projects/amorfy/) para ver a diferença entre ferramenta utilitária e fluxo guiado.
3. Releia [Lógica de Programação](/pt/reference/logica-de-programacao/) pensando em regras de sorteio e estados de interação.
