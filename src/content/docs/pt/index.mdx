## O que é este portal

Este é um portal vivo de engenharia: um lugar único para **como as coisas funcionam**, **por que decisões foram tomadas** e **como entregar rápido sem caos**.

Use quando precisar:

- **Uma fonte de verdade** para configuração e manutenção de projetos
- **Registros de decisão** que explicam compensações (para você não reabrir debates)
- **Guias** que você aplica hoje (checklists, comandos, copiar/colar)
- **Padrões** que mantêm velocidade alta e qualidade consistente

<Aside type="note" title="Princípio">
Se um eu do futuro pode perder **30 minutos** com algo de novo, isso merece documentação.
</Aside>

- Deixo visível uma seção de **"Próximas ações"** nas páginas de projeto
