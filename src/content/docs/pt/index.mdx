## O que é este portal?

Este é um repositório de referências técnicas para programadores iniciantes e profissionais.
Aqui você vai encontrar os melhores repositórios do Github, as melhores APIs e tudo o que você precisa para aprender e se especializar
no mundo do desenvolvimento de software.

Use quando quiser:

- Se atualizar sobre as novidades da nossa área.
- Buscar referências de repositórios interessantes e APIs públicas.
- Se informar sobre os melhores livros da nossa área.
- Comprar produtos que programadores adoram.

<Aside type="note" title="Dica">
A melhor maneira para se destacar em uma área, é estando cercado de informações sobre ela.
</Aside>
